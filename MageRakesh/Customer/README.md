# Module Name : MageRakesh_Customer
# Module Description

1. **Step 1**

> This module is created for assigning customer Status from frontend or backend section

# Module Installation Instruction and requirement

1. **Step 2**
>This module is tested on  2.3.5-p2 version with default magento luma theme

>Run following command one by one : 

> sudo php bin/magento module:enable MageRakesh_Customer
>
>sudo php bin/magento ma:en;
> 
>sudo php bin/magento s:up;
> 
>sudo php bin/magento s:d:c;
> 
>sudo php bin/magento s:s:d -f;
> 
>sudo php bin/magento c:f;
> 
>sudo php bin/magento c:cl;
> 
>sudo php bin/magento ma:dis;
> 
>sudo chmod 777 -R var/ generated/ pub/

# Module Features

1. **Frontend**

> Login to magento 2 customer section first and then you will be able to see the (Go To Status) button in customer menu section.

>After that you will be able to see the customer status section here and you can update you status from here

> Your status will appear on top header of welecome message section
2. **Backend**
>Open the customer edit section and in the bottom page you will able to see dropdown box of customer status

> You can update your status from here when you update your customer
